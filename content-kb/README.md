# FITPEAK コンテンツ・ナレッジベース（伸びる台本ライブラリ）

FITPEAK（Taku｜AIで筋トレを科学する）のショート動画台本を、**生成→批評→改善**のループで磨き、
合格したものだけを蓄積し続ける場所。将来のコンテンツ量産エンジンの学習元になる。

## 仕組み（R&Dループ）
1. テーマを選ぶ（`fitpeak-tiktok`スキルの`topic_categories`／直近と被らせない）
2. フック候補を複数出し、最も刺さるものを選ぶ
3. v1台本を書く → `rubric.md`で採点 → 弱点を特定 → v2へ改善（必要ならv3）
4. 加重スコアが**合格ライン（8.0）以上**になったら `scripts/` に保存
5. その回で効いた打ち手を `playbook.md` に蒸留（次回以降の品質が複利で上がる）
6. `log.md` に試行錯誤の記録を残す

## フォルダ
- `rubric.md` — 伸びる台本の採点基準（品質ゲート）
- `playbook.md` — 蓄積された「効く型」。回すたびに更新（最重要資産）
- `scripts/` — 合格台本。各ファイルにメタ情報＋HeyGen貼り付け用全文＋スコア＋勝因
- `log.md` — 各ラウンドの試行錯誤ログ

## 制作標準
台本フォーマット・トーン・ハードルール（音声用は漢字カナ開き・3ゾーンレイアウト・中国語字形禁止・
話者SNS TAKUMA／Ta U・AI開示）は `fitpeak-tiktok` スキルに従う。本KBはその「中身（何を言うか）」の質を上げる。
